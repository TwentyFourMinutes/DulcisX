    Introduction

    Welcome to the documentation, there are a few requirements your skills should meet.

    • Pretty solid understanding of C#, if not take a look at the C# Programming Guide

    • Basic understanding of how Visual Studio Extensions work, otherwise you should take the First Extension Guide which will teach you some basics.

    In the guides section you can find information on how to install DulcisX and articles which help you to get started.

    You need more help?

    Feel free to create a GitHub issue or join the Discord Server .
